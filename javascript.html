Fundamentos de Javascript

El scope puede definirse como el alcance que una variable tendrá en el código. El scope decide a que variables tienes acceso en cada parte del código. Exiten dos tipos de scope, el scope global y el scope local.

Un closure es una función que recuerda el entorno en el cual fue creada. Una función interna definida en una función externa.

Un callback es cuando pasas una función como parámetro a otra función y que esa función se ejecute

Dentro del HTML escrimos las etiquetas <script> </script> y dentro de ellas escribimos nuestro código javascript.

Con document.write escribimos un mensaje en el html.

Con alert envía una ventana emergente en el navegador con un mensaje en pantalla ejemplo: alert ("Hola Mundo!")

Para concatenar una cadena de texto (string) con una variable o función utilizamos el signo de $ seguido por llaves {} ejemplo definimos la variable nombre: console.log (`Hola mi nombre es ${nombre}`) Si la variable esta definida dentro de una función sólo se mantienen con vida dentro de la función.

Podemos crear una variable con let o para definir una varibale que no va a cambiar o que no se va asignar un nuevo valor utilizamos const

Utilizando prompt podemos pedirle un dato al usuario mediante una ventana, ejemplo: nombre = prompt(`¿Cuál es tu nombre?`)

Con la etiqueta parseInt convertimos un valor string a un valor numérico, también podemos convertir un valor flotante(un número con decimal) a un valor entero, con parseFloat convertimos el valor en flotante(un número con decimal) y con toFixed( ) hacemos que salgan la cantidad de decimales que escribimos dentro del parentesis.

Algunas funciones para redondear números son:

  • Math.floor redondea hacia abajo 3.9 = 3
  • Math.ceil redondea hacia arriba 3.1 = 4
  • Math.round redondea dependiendo 3.1 = 3, 3.5 = 4
  • Math.random() regresa un numero aleatorio entre 0.01 y 0.99

Para definir una fecha utilizamos Date(año, mes(restandole 1), dia)

Estructura de objetos en JavaScript const nombreObjeto = { // inicia la construcción del objeto
// Los objetos tienen atributos que apuntan a un valor
atributoA: 10, // Para detonar un atributo, se escribe el nombre del atributo, dos puntos ":" y el valor
 atributoB: 20, // Para escribir varios atributos, se separan por coma ","
 atributoC: 30 // El último atributo no lleva coma ni punto y coma
 } // Cierra la consutrucción del objeto

Métodos en los objetos son una función que vive dentro de un objeto. const myObject = {
atributoA: "nombre",
atributoB: 23,
metodoA(newAtributoA) { // creación del metodo que recibe el parametro newAtributoA
this.atributoA = newAtributoA // bloque de codigo que ejecuta el metodo } }

Eventos con addEventListener vamos a detectar eventos que se ejecutan cuando el usuario realiza algo, addEventListener requiere de dos parametros, ejemplo: document.addEventListener('keydown', presionarTecla)

  • keydown ejecuta el evento cuando se presiona una tecla del teclado
  • keyup ejecuta el evento cuando se presiona y suelta la tecla
  • mousedown se ejecuta cuando se presiona el click del mouse
En la función presionarTecla() dentro de los parentesis escribimos un parametro (evento) para ver los detalles que internamente envia addEventListener a la función, ejemplo agregamos lo siguiente al código de arriba: function presionarTecla(evento) {
console.log(evento) }

Descargar de la siguiente página Nodejs

Instalamos nuestro servidor web con express para instalar express entramos al siguiente sitio web y buscamos el paquete, podemos utilizar el siguiente comando en terminal para instalarlo npm install express este comando se ejecuta dentro de la carpeta local donde guardaremos el archivos que va aejecutar el siguiente código:

var express = require('express')

var app = express()

//Esta es nuestra ruta Home
app.get('/', function (req, res) {
res.send('Hello World')
})

app.listen(3000)
